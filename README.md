# Text-based Emotion Classifier

## Dataset 
### Source of Dataset
> https://www.kaggle.com/praveengovi/emotions-dataset-for-nlp by Praveen

### Format of Dataset
> | text         | emotion |
> |--------------|---------|
> | I find it... | sadness |

> **Note:** ***text*** and ***emotion*** are separated by a semi-colon ***';'***.
